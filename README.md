# Kanban Board
Task tracking and organizing board.

## Usage ##
> Webpack compile 'npm run react-dev'
```
webpack -d --watch
```

## Requirements ##
An `nvmrc` file is included if using [nvm](https://github.com/nvm-sh/nvm).
* Node v14.2.0
* etc

## Development ##
From within the root directory:
```
npm install
```
